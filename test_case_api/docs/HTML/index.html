Quick Start

Spin up the API and produce your first test case in minutes. Commands assume Windows PowerShell with Python 3.8+ and Ollama installed locally.

cd test_case_api
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
$env:OLLAMA_MODEL = "mistral"
python app.py
curl http://localhost:5000/health
curl -X POST http://localhost:5000/generate `
  -H "Content-Type: application/json" `
  -d @samples/single_requirement.json
By default the application calls http://localhost:11434/api/generate. Update OLLAMA_BASE_URL or OLLAMA_MODEL in .env to target remote endpoints or different LLM variants.
