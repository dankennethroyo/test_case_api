  â€¢ Generates detailed, executable test case
  â€¢ Focuses on system-level, black-box testing

  - Test Steps: Numbered actions with expected results
  - Expected Result: Final verification
  - Test Data: Specific values & parameters
  - Edge Cases: Boundary conditions
